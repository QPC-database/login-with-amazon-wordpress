Login with Amazon for Wordpress

== Description ==

A simple WordPress extension that lets you add a Login with Amazon button to your site.

== Installation ==

Download the plugin as a zip file [here](https://github.com/amzn/login-with-amazon-wordpress).

In your WordPress admin console, upload and activate the plugin under: Plugins -> Add New -> Upload 


== Usage ==

You can go to login.amazon.com (follow the steps in the documentation) to register your website. You will get a client id there.
After that follow the following steps:
1. Download the plugin.
2. Log in to your WordPress control panel.
3. In the side bar select Plugins: Add New
4. From the top of the page select Upload Plugin.
5. Upload the file you downloaded in step 1 to the form you have navigated to.
